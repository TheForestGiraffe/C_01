## Skills & Concepts

Throughout this module you practice:

- Basic pointer usage and dereferencing  
- Pointer to pointer patterns  
- Manipulating values *by reference*  
- Simple array handling  
- Writing functions to match strict prototypes  

This builds essential understanding of memory and references in C, which is fundamental for all higher-level C work.
